# Nabes Style Ventures Website

A modern, responsive, and interactive website for Nabes Style Ventures (Accra, Ghana). Green and white theme, showcasing home detergents and sewing materials.

## Project Structure
- index.html
- assets/
  - styles.css
  - script.js
  - images/ (place your images here)

## Getting Started
1. Open index.html in your browser to preview.
2. Replace placeholder images in assets/images with your own:
   - logo.png
   - hero-products.png
   - liquid-soap.jpg
   - floor-cleaner.jpg
   - thread-set.jpg
   - needles.jpg
   - about.jpg
3. Update contact information in index.html:
   - Phone, Email, and social media links.
4. Add more products:
   - Duplicate a product card in the Products section.
   - Set data-category to 'detergents' or 'sewing'.
   - Update product name, description, price, and image.

## Features
- Mobile-friendly navigation.
- Product category filters.
- Mini cart (client-side, persists in localStorage).
- FAQ accordion.
- Contact form with basic validation.

## Deployment
- GitHub Pages: push this folder to a repo and enable Pages.
- Netlify/Vercel: drag-and-drop deploy or connect your repo.
- Any static hosting will work.

## Customization
- Colors are defined in assets/styles.css under :root (green shades).
- Fonts use Google Fonts (Poppins) in index.html.
- Add pages or sections as needed.
